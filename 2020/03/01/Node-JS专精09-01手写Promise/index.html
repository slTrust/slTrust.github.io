004用户怎么用Promise

Promise的完整API是什么

Promise是一个类

  • JS里类实际就是个函数
  • 类属性 length (可忽略) 因为我们知道它只接受一个参数
  • 类方法：all / allSettled / race / reject / resolve
  • 对象属性：then(重要) finally / catch
  • 对象内部属性： state = pending / fulfilled / rejected
