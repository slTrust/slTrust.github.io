Node-web01_00curl命令使用

curl

  • curl -s -v -o nul 网址
    • -s silent,隐藏进度条
    • -v 打印全部的header
    • * 开头是注释
    • > 是 http请求
    • < 是 http响应
    • -o nul 隐藏 HTML文本
      • linux 或者 mac需要把 nul 改成 /dev/null
      • curl -s -v -o null 网址
