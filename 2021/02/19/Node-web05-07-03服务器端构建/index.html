创建数据库表

  • yarn typeorm:build

解决报错

设置环境变量，服务器是 production

NODE_ENV=production 后，yarn install 不会install dev的东西

  • 你要这样 yarn install --production=false

docker的--network=host选项

  • 让容器共享你 阿里云机器的IP，这样就可以不做端口映射
  • 数据库连接不上问题
    • 容器的localhost 不是你本机的 localhost
