作用域问题

除了全局作用域和函数作用域,es6新增了块级作用域，即用{ }包裹的代码块。

块级作用域

内层变量和外层变量的值互不影响。内层无法读取外层，外层无法读取内层。

const 常量

特性

  • 声明时就需要赋值，否则报错；一旦声明不可重新赋值，否则报错；
