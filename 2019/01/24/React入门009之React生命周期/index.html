此时看React版的生命周期(完整版)

在线代码

  • constructor 创建
  • componentWillMount 将要挂载
  • render 填充/更新
  • componentDidMount 挂载完毕
  • componentWillUpdate 更新之前
  • shouldComponentUpdate 要不要更新
  • componentDidUpdate 更新之后
  • componentWillUnmount 没死之前
  • componentWillReceiveProps 父组件传递props了

我们在页面一般做什么？

  1. ajax请求数据
  2. 更新数据 setState
  3. 事件监听 onClick
  4. 初始化state
