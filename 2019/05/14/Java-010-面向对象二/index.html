多态

  • 父类对象指向子类实例

如下例子 Man / Dog /Car 都会跑

  • 如果单独调用自己的 manRun/dogRun/carRun 就很重复和耦合
  • 而使用多态就可以只用一个 objRun 此时只要传递实现类 调用即可
