get请求是没有body的吗？

  • googleget payload

  • get请求一般在生产环境里是没有payload的

  • get请求是不禁止有 payload的
  • 只是 http 中明确规定了不推荐

控制get参数是否必填

  • 通过注解设置参数是否必填@RequestParam(value="charset",required = false)

get请求通常不能用来传递敏感信息，如登录。

幂等性

无论多少次get请求 都应该是一种结果

不同于 post/put/delete等

RESTful API

  • 使用HTTP动词代表动作
    • GET 获取资源
    • POST 新建资源
    • PUT 更新资源
    • DELETE 删除资源
  • 使用 URL（名词）代表资源
    • 使用复数来代表资源列表
    • 资源里面没有动词
