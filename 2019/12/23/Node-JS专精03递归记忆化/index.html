React.memo 减少计算

解析

  • 如果你用函数式组件，那么每次setN 都会导致执行 Child() 里的打印
    • 如果 Child() 里有定义变量，那么也会重新执行，非常浪费
  • 优化就是使用 React.memo减少计算 记忆化组件(把结果缓存起来)
