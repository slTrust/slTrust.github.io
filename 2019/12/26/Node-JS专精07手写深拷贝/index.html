JSON序列化和反序列化

JSON序列化反序列化的缺点是什么

  • 不支持函数(会被忽略)
  • 不支持 undefined 只支持null

  • 不支持引用(循环引用)

  • 不支持 Date ,JSON.stringify 之后会是 Date的字符串形式

  • 不支持正则表达式
  • 不支持 Symbol
