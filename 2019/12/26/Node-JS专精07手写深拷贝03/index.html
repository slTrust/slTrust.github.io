回到答题思路

五步

  • why 要解决什么问题
  • how 如何解决的
  • pros 优点：只要是你自己实现的基本没什么优点，唯一能证明就是 js基础掌握的较好，因为一般都是用第三方的库，因为第三方的库测试比你完善
  • cons 缺点
    • 如果你的类型超过了我的 if else 我就搞不定了，比如 set / map
  • more 解决缺点的方案
    • 直接用第三方库 比如 lodash / immutable.js
