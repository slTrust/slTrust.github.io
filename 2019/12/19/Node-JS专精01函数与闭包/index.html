函数是什么？

  • 最开始程序员写的大部分是汇编，后来有了 C / C++ 那时候没有函数，只有子程序
    • 比如有四行代码完成特定的功能
    • 给这四行代码起了一个名字 叫做函数
    • 但是那个时候不叫 函数而叫 子程序

子程序

  • 一个或多个语句组成
  • 完成特定功能
  • 功能相对独立

子程序又分三种

  • 函数(有返回值) function
  • 过程(无返回值) procedure
  • 方法 (类或对象中) method

JS 里只有函数和方法

因为JS的所有函数都有返回值，你不写return 但是返回值是 undefined

数学领域里的函数

  • 数学中的函数
    • 两个集合之间的对应关系 如y = f(x) 比如方程式 y = a*x + b
      • X 叫做 定义域 Y 叫做 值域
    • 定义域里的每个元素都对应一项 值域中的元素
  • f()在数学领域是不合法的，因为没有输入
  • 函数式编程
    • 如果你的函数符合数学函数的定义，就是 函数式的

但是 JS的函数不是数学函数

最大的区别就是，你给我一个值我可能给你返回不同的结果

我们平时用的是哪个函数?

是编程里的子程序，还是数学里的函数？

  • 答案是 子程序

函数的返回值由什么确定？

影响因素

  • 调用时输入的参数 params
  • 定义时的环境 env

为什么不是 ‘x2’?

  • x 很好理解是 参数 所以 f1('x') 时x = 'x'
  • a 在这里不是参数 ，它是环境，它由 f1 所处的环境确定的
    • 如果你是参数 则在调用时 确定

如何打印成 0 1 2 3 4 5呢？

  • 改用 let
  • 立即执行函数

为什么？

  • 用 var 声明的时候 用的都是 一个 i，每个闭包用的同一个 i
  • 用了 let 后，每个闭包分配一个新的 i

闭包的特点

  • 能让一个函数维持住一个变量
  • 但不能维持这个变量的值
  • 尤其是变量的值会变化的时候

对象是穷人的闭包

  • 对象也可以来维持一个变量
  • 如果一门语言不支持闭包，你可以用对象代理
