箭头函数不支持this

箭头函数如何处理 a ,就如何处理 this

箭头函数把 this 当作外部的变量，仅此而已，但是非箭头函数的 this 有很多特殊处理

箭头函数不支持this 指的就是尖头函数对 this 和其他变量一视同仁，不会特殊对待

非箭头函数的this

网传死记方法

  • this是上下文
  • 全局环境执行时候,this是全局对象
  • 调用对象方法时，this是该对象
  • 函数里调用函数时，this是全局对象
  • 箭头函数的this，不看调用，看定义(静态作用域)
  • 还有人说箭头函数里的 this 指向外面的this
  • 用new调用函数时，this是新增对象
  • 可以用call/apply/bind指定this
