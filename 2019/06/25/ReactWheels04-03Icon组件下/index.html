让Icon支持sass

icon.scss

在 icon.tsx 里 import ‘./icon.scss’

运行 yarn start ，报错了

安装loader,修改webpack配置

  1. sass-loader 将你的 icon.scss读取到内存(字符串) 把 sass语法变成 icon.css
  2. css-loader 将 icon.css 变成一个对象
  3. style-loader 将 对象 变成 一个 style标签 把对象里的内容放到 style.innerHTML里
