js jsx ts tsx的区别

  • jsx 是 js 的扩展，支持 用 js 的方式写 标签 是xml标准

  • js 和 ts 的区别就是 ts = Type + JavasSript 也是对js的扩展

  • tsx 是对 ts 的扩展，支持 写标签 支持 Type

所以 tsx 是最厉害的
