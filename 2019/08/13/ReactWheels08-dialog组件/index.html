一个 z-indexBug

  • 场景如下
  • 你可能会想给我们的 dialog 设置 z-index:9999
    • 如果是平级那没问题
    • 但如果你的 Dialog 在一个容器里那个容器的 z-index:9 你就算是 z-index:99999 也没用

解决办法就是：不要出现在任何元素的里面

解决方案是 react portal

  • react portal
  • react portal中文
  • react protal知乎

推荐dialog的 z-index 设置为1 ，因为用户可以改啊！

z-index 结构划分

为什么推荐我们的 dialog z-index:1呢？

  • 因为层级低容易出现，这样别人用的时候再去覆盖 fui-dialog{z-index:100}
